# Development Roadmap

- **Phase 1 — Foundation (complete):** monorepo, UI shell, typed API, persistence plumbing, observability, tests, CI, and containers.
- **Phase 2A — Secure Dataset Ingestion (complete):** streamed CSV upload, governed storage, technical metadata, previews, library, and archive flow.
- **Phase 2B — Schema Inference and Intelligent Column Mapping (next):** non-mutating schema proposals, explicit roles, and user confirmation.
- **Phase 3 — Probabilistic Forecasting:** time-based evaluation, baselines, probabilistic models, and forecast APIs.
- **Phase 4 — Trust and Conformal Calibration:** calibrated intervals, drift monitoring, model cards, and diagnostics.
- **Phase 5 — Causal Intelligence:** causal assumptions, incrementality estimation, sensitivity analysis, and evidence.
- **Phase 6 — Scenario Simulation:** versioned scenarios, uncertainty propagation, and comparison workflows.
- **Phase 7 — Risk-Aware Optimization:** explicit constraints, robust objectives, and human-approved allocations.
- **Phase 8 — RAG Copilot and Agent Orchestration:** grounded retrieval, citations, deterministic tools, and approval gates.
- **Phase 9 — Deployment and Demo Hardening:** production data services, authentication, telemetry, scaling, and operational runbooks.
